# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 4.x     | :white_check_mark: |
| 3.x     | :white_check_mark: |
| < 3.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in SmartRAG, please report it responsibly:

1. **Do not** open a public GitHub issue for security vulnerabilities.
2. Email **b.yerlikaya@outlook.com** with:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Suggested fix (if any)
3. Allow reasonable time for a fix before public disclosure.

We appreciate your efforts to report vulnerabilities responsibly.
